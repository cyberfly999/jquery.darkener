jQuery.darkener

A jQuery-Plugin to superimpose gradients over any container

Color Manipulation 1 Color Manipulation 2 Color Manipulation 3 Color Manipulation 4 Part Overlay 1 Part Overlay 2

Color Manipulation 1

↑

One gradient overlay

			
$('CONTAINER').darkener({
	align: 'bottom',
	colorStart: 'rgba(0,0,255,0.5)',
	colorEnd: 'rgba(255,255,0,0.8)',
	size: '100%',
	mixBlendMode: 'difference',
	opacity: '1'
});
